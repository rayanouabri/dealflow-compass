  SECRETS SUPABASE — Configuration AI (Gemini ou Vertex AI)
═══════════════════════════════════════════════════════════════

Dashboard: https://supabase.com/dashboard/project/anxyjsgrittdwrizqcgi/functions/secrets

Pour chaque secret : "Add Secret" ou "Edit".

───────────────────────────────────────────────────────────────
OPTION 1 : GEMINI (par défaut, gratuit)
───────────────────────────────────────────────────────────────
1. AI_PROVIDER = "gemini" (ou laissez vide, c'est le défaut)

2. GEMINI_KEY_2 (ou GEMINI_API_KEY)
   Clé API Google Gemini.
   Obtention: https://makersuite.google.com/app/apikey (gratuit)

3. GEMINI_MODEL (optionnel, défaut: "gemini-2.5-pro")
   Modèles disponibles:
   - gemini-2.5-pro (le plus puissant, recommandé)
   - gemini-2.0-flash (rapide)
   - gemini-pro (puissant)
   - gemini-1.5-pro (très puissant)
   - gemini-1.5-flash (rapide et efficace)

───────────────────────────────────────────────────────────────
OPTION 2 : VERTEX AI (Google Cloud)
───────────────────────────────────────────────────────────────
1. AI_PROVIDER = "vertex"

2. VERTEX_AI_PROJECT_ID
   ID du projet Google Cloud (ex: "my-project-123456")

3. VERTEX_AI_LOCATION (optionnel, défaut: "us-central1")
   Région (ex: "us-central1", "europe-west1")

4. VERTEX_AI_MODEL (optionnel, défaut: "gemini-pro")
   Modèles disponibles:
   - gemini-pro
   - gemini-1.5-pro
   - gemini-1.5-flash

5. VERTEX_AI_CREDENTIALS (JSON)
   Service account JSON avec permissions Vertex AI.
   Format: {"type":"service_account","project_id":"...","private_key":"...",...}

───────────────────────────────────────────────────────────────
3. BRAVE_API_KEY (requis dans tous les cas)
───────────────────────────────────────────────────────────────
Clé API Brave Search. Utilisée pour les couches de recherche web.

Obtention: https://brave.com/search/api/

Exemple (à remplacer par votre clé):
BSAjI6tJ9s5t2qMZZYNTtBDxHQhqVFJ
